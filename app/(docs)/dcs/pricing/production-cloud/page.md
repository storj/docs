# Production Cloud Pricing

{% callout type="info" %}
**Production Cloud will be discontinued as a bundled offering on July 1, 2026**. The individual components that make up Production Cloud, including Object Storage and Object Mount, will be still available as stand-alone products. Customers who need both can purchase [Object Storage at the Standard or Advanced tier rate](docId:1ea1de87-5e14-402b-afb7-15d83c135563) and add [Object Mount licenses](docId:b464fda8-9e3b-4417-aa64-2eb1efb69c39) separately. This gives you more flexibility to size each component to your actual usage rather than committing to a fixed bundle.

- You will need to Opt-In for the [new pricing](docId:1ea1de87-5e14-402b-afb7-15d83c135563) or Opt-Out. An opt-in/out selection is in your [](docId:nGzxQBhV8nx5Pukj6O0zT). Make your selection before July 1, 2026.
- **Existing Projects** created before November 1, 2025 will continue to use the current [](docId:x3xPZJXCp!CB@spN) until July 1, 2026.
- **New Projects** created after November 1, 2025 will continue to use the current [](docId:mqRRgT,hL*dk3zNT) until July 1, 2026.
- **New Projects** created on or after July 1, 2026 will be charged using the new [](docId:1ea1de87-5e14-402b-afb7-15d83c135563).

If you opt-out or do nothing, **your account will be frozen and data deleted within 45 days**.
{% /callout %}

## Overview

Storj’s Production Cloud offerings combine the best of global and regional workflows with the power and convenience of local file system access.

Production Cloud pairs **Object Mount** with our **Distributed Cloud Storage** for better performance and added value for every production team.

- Access and edit your media in real time, with consistently low latency
- Works with all your existing applications
- No proprietary formats

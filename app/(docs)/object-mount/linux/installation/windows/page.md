Storj provides a full-featured, native version of [Object Mount for Windows](docId:TSMB5yXSc4FcqwXj).

However, if you would like to install the Linux version of Object Mount within your Windows environment, you can leverage üåê [Windows Subsystem for Linux 2 (WSL2)](https://learn.microsoft.com/en-us/windows/wsl/install) or inside a Linux virtual machine or container.

## Windows Subsystem for Linux

WSL installs Ubuntu by default. 

For Ubuntu, follow either the [glibc Scripted Installer](docId:ahWohd5eegh6eizi) instructions or the [Debian Package Installer](docId:aemie9zeiP9Nie2k) instructions.

Once completed, you will be able to use Object Mount with any Linux applications also installed in the Linux subsystem.
